# A360-Labs Landing Page

A360 Labs Landing Page
A360 Labs is a modern digital agency specializing in AI visuals, web development, brand identity, vector illustrations and content design. This repository contains the desktop version of the landing page, designed for creative agencies, tech start-ups, or portfolio showcases.

Overview
The landing page communicates innovation, creativity, and full-spectrum digital solutions. It includes:
    Hero Section: Headline, subtext, call-to-action button, and hero imagery
    Feature Section: Four key service highlights
    Testimonial Section: Client feedback or quotes
    Call-to-Action (CTA) Section: Encourages user engagement
    Footer Section: Company info, copyright, and links

The page is designed for a desktop aspect ratio (for now), with dark futuristic aesthetics and electric blue highlights.

Features
    Clean, minimal, and futuristic UI
    Modular sections for Hero, Services, Testimonials, CTA, and Footer
    Easy-to-customize content and imagery
    Compatible with modern web frameworks

Repository Structure
    assets/ – Logos, icons
    img/ - Images
    css/ – Stylesheets
    index.html – Main landing page
    README.md – Project documentation

Technologies Used
    HTML5
    CSS3 (Flexbox/Grid)
